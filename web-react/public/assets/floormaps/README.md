# Floor Maps Directory

This directory contains PDF floor maps for campus buildings.

## Science Hall Floor Maps

Place the `science-hall.pdf` file in this directory.

The PDF should contain multiple pages, with each page representing a different floor of the Science Hall building.

### Expected Format:
- Page 1: Ground Floor
- Page 2: First Floor
- Page 3: Second Floor
- etc.

### How to Add:
1. Copy your Science Hall PDF to this location
2. Rename it to: `science-hall.pdf`
3. The floor map viewer will automatically display all pages

### Path Reference:
The app expects the file at: `/assets/floormaps/science-hall.pdf`
